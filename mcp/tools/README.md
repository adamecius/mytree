# MCP tools

## Why this module exists
This folder groups callable tool primitives used by orchestration layers (API, MCP server, future agents).

## Files
- `retrieve_tool.py`: placeholder retrieval utility.
- `graph_tool.py`: graph neighborhood helper.

## Function one-liners
- `retrieve(query, k)`: returns placeholder top-k retrieval contexts.
- `neighbors(graph, node)`: returns sorted immediate neighbors for a node.
